Figma Converter for Unity v5.5.2

To install (run once per machine / after clone):
  In Unity menu: LakbayTala → Install Figma Converter (v5.5.2)
  Or: Assets → Import Package → Custom Package → select FigmaConverter_Unity_v5.5.2.unitypackage

Package is stored here so the project can install it anytime without re-downloading.
