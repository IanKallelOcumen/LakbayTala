LilitaOne (default UI font)
- LilitaOne-Regular.ttf is in this folder. Unity imports it as a legacy Font (UI.Text).
- Legacy: In Font Replacer, click "Use LilitaOne" — it assigns this TTF for all UI.Text.
- TextMeshPro: Right-click LilitaOne-Regular.ttf in Project → Create → TextMeshPro → Font Asset.
  Save as "LilitaOne SDF" (here or in Resources/Fonts). Then "Use LilitaOne" will assign it for TMP too.
- Apply everywhere: LakbayTala → Font Replacer → "Use LilitaOne" → "Replace fonts in ALL Scenes & Prefabs".
